# DemoBlaze Test Automation Project

## Project Description
This project contains automated tests for DemoBlaze website using:

- Java
- Selenium WebDriver
- TestNG
- Maven

## Test Coverage

- Product Tests
- Cart Tests
- Checkout Tests
